# Tic Tac Toe (C - Terminal) 🎮

A simple Tic Tac Toe game implemented in C for the terminal.

## Features
- Two-player mode (X vs O)
- Win/draw detection
- Basic terminal UI

## How to Run
1. Clone the repo: `git clone <your-repo-url>`
2. Compile: `gcc tic_tac_toe.c -o tic_tac_toe`
3. Run: `./tic_tac_toe`

## Code Structure
- `tic_tac_toe.c`: Main game logic
